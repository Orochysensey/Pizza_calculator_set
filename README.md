# 🍕 Калькулятор выгодности пиццы

**Приложение для Android**, которое поможет вам узнать, какая пицца выгоднее по соотношению цены к площади.

## ✨ Возможности

- 🍕 **Расчёт одной пиццы** — введите диаметр и цену
- 📦 **Расчёт сета** — добавьте несколько пицц, укажите их размеры и стоимость
- 📊 **История расчётов** — сохраняются последние 10 результатов
- 🎨 **Смена фона** — разные фоновые изображения для режимов "Одна пицца" и "Сет"

## 📲 Установка

1. Скачайте APK из раздела [Releases](https://github.com/Orochysensey/Pizza_calculator_set/releases)
2. Разрешите установку из **неизвестных источников** (на телефоне)
3. Откройте файл и нажмите **Установить**
